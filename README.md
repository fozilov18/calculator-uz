# Calculator UZ 📱

Ushbu loyiha HTML, CSS va JavaScript yordamida yaratilgan zamonaviy onlayn kalkulyatordir.

## 🚀 Jonli havola
Loyihani ko'rish uchun bu yerga bosing: [uz-kalkulyator.netlify.app](https://uz-kalkulyator.netlify.app)

## ✨ Imkoniyatlari:
*   Asosiy matematik amallar (+, -, *, /)
*   Xatoliklarni avtomatik aniqlash (Try-Catch)
*   Zamonaviy va qulay dizayn
*   Mobil qurilmalar uchun moslashuvchan (Responsive)

## 🛠 Ishlatilgan texnologiyalar:
*   HTML5
*   CSS3 (Grid & Flexbox)
*   JavaScript (ES6+)
